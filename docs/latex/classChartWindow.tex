\doxysubsection{Detailed Description}
Klasa \doxylink{classChartWindow}{Chart\+Window} reprezentuje okno wykresów w aplikacji Project FOKA. 

Wyświetla dane z sensorów w formie wykresów liniowych. Każdy sensor ma swój osobny wykres, aktualizowany przy nadejściu nowego pakietu danych.
