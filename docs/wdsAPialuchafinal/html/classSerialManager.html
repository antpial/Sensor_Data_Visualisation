Detailed Description

Klasa odpowiedzialna za odbieranie danych z portu szeregowego oraz ich przetwarzanie.

SerialManager nasłuchuje portu szeregowego, odczytuje linie danych, parsuje je do struktury ParsedPacket i emituje sygnały w przypadku odebrania danych lub błędu.
